# gprp-sorter
a gRPC demo written in Go

## Usage

1 clone to you `$GOPATH/src`

```shell script
git clone git@github.com:chasecs/grpc-sorter.git 
```

2 run the server

```shell script
go run sorter_server/sorter_server.go
```

3 run the client on another terminal

```shell script
go run sorter_client/main.go
```
